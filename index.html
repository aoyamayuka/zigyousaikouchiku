~公募要領まとめ~

【公募期間】

  • 公募開始：令和3年7/30
  • 申請受付：令和3年8月下旬
  • 応募締切：令和3年9/21　18時

【申請に必須な要件】

  1. 売上が減っている
    • コロナ前（2019年-2020年3月）とコロナ後（2020年４月以降）の同月を比較して10％（202010月以降は5％）減少している
    • 比較月は連続する6ヶ月のうちの任意の3か月を選択すること
    • 例：10月、3月、4月→×（10、11、12、1、2，3の連続する6ヶ月からあぶれるため4月は選択できない）
  2. 新分野展開、業態転換、事業・業種転換、事業再編に取り組む
  3. 認定支援機関と3ｰ5年の事業計画を作成する
    • 補助金額3000万円を超える場合には金融機関も参加して策定する
      • 　
    • 事業計画終了後に付加価値額年率平均3.0％（グローバルは5.0％）以上増加すること

【補助対象事業の類型を詳しく見る】

①通常枠

  1. 対象
    • 中小企業（個人事業主含む）
    • 中堅企業
  2. 補助金額
    • 【従業員数 20人以下】100万円～4,000万円
    • 【従業員数 21~50人】 100万円～6,000万円
    • 【従業員数 51人以上】 100万円～8,000万円
  3. 補助率
    • 中小企業者等　2/3 (6,000万円超は1/2)
    • 中堅企業等　 1/2 (4,000万円超は1/3)
  4. 補助事業実施期間
    • 交付決定日から12ヶ月以内（採択発表日から14か月後の日まで)

②大規模賃金枠

  1. 対象
    • 従業員数101人以上の中小、中堅企業
  2. 補助金額
    • 【従業員数 101人以下】8,000万円～1億円
  3. 補助率
    • 中小企業者等　2/3 (6,000万円超は1/2)
    • 中堅企業等　 1/2 (4,000万円超は1/3)
  4. 補助事業実施期間
    • 交付決定日から12ヶ月以内（採択発表日から14か月後の日まで)
  5. 備考
    • 全公募で150社限定
    • 事業終了後、要件を達成できなかった場合には補助金の返還あり

③卒業枠

  1. 対象
    • 中小企業
  2. 補助金額
    • 6,000万円～1億円
  3. 補助率
    • 中小企業者等　2/3
  4. 補助事業実施期間
    • 交付決定日から14ヶ月以内（採択発表日から16か月後の日まで)
  5. 備考
    • 全公募で400社限定
    • 事業終了後、中堅、大企業に成長させる要件あり
      >→要件未達成の場合は補助金の返還あり
    • 不採択の場合には、通常枠で再審査

➃グローバルV字回復枠

  1. 対象
    • 中堅企業
  2. 補助金額
    • 中堅企業者等　1/2
  3. 補助率
    • 中堅企業 1/2
  4. 補助事業実施期間
    • 交付決定日から14ヶ月以内（採択発表日から16か月後の日まで)
  5. 備考
    • 全公募で100社限定
    • 事業終了後、付加価値額の年率平均5.0%以上増加させる要件あり
      →要件未達成の場合は補助金の返還あり
    • 売上減少要件の代替案不可
    • 売上減少要件、10％以上減少×→15％以上減少
    • グローバル枠で不採択の場合には、通常枠で再審査

⑤特別枠

  1. 対象
    • 中小企業
    • 中堅企業
  2. 補助金額
    • 【従業員数 5人以下】100万円～500万円
    • 【従業員数 6~21人】 100万円～1,000万円
    • 【従業員数 21人以上】 100万円～1,500万円
  3. 補助率
    • 中小企業 3/4
    • 中堅企業 2/3
  4. 補助事業実施期間
    • 交付決定日から12ヶ月以内（採択発表日から14か月後の日まで)
  5. 備考
    • 枠に限りあり、今回で終了の可能性あり
    • 特別枠で不採択の場合、通常枠で再審査

  1. 対象
    • 中小企業
    • 中堅企業
  2. 補助金額
    • 【従業員数 5人以下】100万円～500万円
    • 【従業員数 6~21人】 100万円～1,000万円
    • 【従業員数 21人以上】 100万円～1,500万円
  3. 補助率
    • 中小企業 3/4
    • 中堅企業 2/3
  4. 補助事業実施期間
    • 交付決定日から12ヶ月以内（採択発表日から14か月後の日まで)

    • 最低賃金枠で不採択の場合、通常枠で再審査あり

【申請に必要な書類】

  1. 事業計画書
  2. 認定支援機関の確認書・金融機関の確認書
    • 補助金額が3000万円を超える場合は金融機関の確認書が必要
  3. コロナ以前に比べて売上高が減少したことを示す書類
    • (a)2020/4~の連続する6ヶ月のうち任意の3か月の合計売上高がコロナ前の同月と比較して10％以上減少している
      (b)2020/10~の連続する6ヶ月のうち任意の3か月の合計売上高がコロナ前の同月と比較して5％以上減少している
      ※コロナ前：2019年‐2020年3月/
    • コロナ前、コロナ後の確定申告書別表一の控え、法人事業概況説明書（法人）
    • コロナ前、コロナ後の確定申告書別表一の控え、青色申告決算書の控え（白色申告書、売上台帳）（個人）
    • ※特例の場合「売上減少の確認に係る特例について」を確認
  4. 決算書
    • 個人事業主は提出なし
    • 決算書の添付ができない場合は収支予算書と企業全体の事業計画書を提出する
  5. ミラサポ「ローカルベンチマーク」
  6. 賃金引き上げ計画の表明書（大規模賃金引上げ枠のみ）
    • 対象賃金台帳も併せて添付
  7. 海外事業の準備状況を示す書類(卒業枠（グローバル展開を実施する場合に限る）、グローバル枠のみ)
  8. 従業員数を示す書類（卒業枠、グローバル枠は不要）
    • 労働者名簿の写し
  9. 場内最低賃金を示す書類 （最低賃金枠のみ）
    • 最低賃金確認書
    • 最低賃金＋30円以内の従業員すべてがわかる賃金台帳
  10. 緊急事態宣言の影響を受けたことの宣誓書（特別枠のみ）
    • 売上減少に係る証明（必要な方のみ）
  11. 固定費が同月に受給した協力金を上回ることの証明書類（加点希望者のみ）
    • 固定費（光熱費、家賃、人件費）が確認できる書類
    • 協力金受給に係る証明書（通帳のコピーまたは協力金確定通知書）
